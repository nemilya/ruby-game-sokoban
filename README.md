Игра SOKOBAN
============

Описание уровня:
http://www.sokobano.de/wiki/index.php?title=How_to_play_Sokoban


    Уровень состоит из элементов:

    Walls: #
    Boxes: $
    Goals: .
    Free squares: ' '
    The Sokoban / the player: @

    Ящики и Сокобан могут находится на "целевой точке":

    Boxes on goals: *
    The Sokoban on a goal: +
